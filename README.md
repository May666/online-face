# online-face
一个基于Springboot和MybatisPlus以及Angular8的在线刷题系统

 最近一直想 \** ，想找个地方刷一刷面试题，可是却没发现一个很方便的能让我开心刷题的地方，
于是就自己想了这么一个系统 起初只是想做一个刷java面试题的系统，后面设计的时候发现可以做成一个通用的刷题系统
，只需要定义类型就可以了
